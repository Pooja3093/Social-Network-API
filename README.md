# Social Network API

## Description

This application was designed as a part of module 18 NoSQL challenge. For this challenge we are supposed to build an API for a social network web application where users can share their thoughts, react to friendsâ€™ thoughts, and create a friend list. The use of Express.js, MongoDB database, and the Mongoose ODM are emphasized.

## Installation

After cloning the repository, run command "npm install" to install all the dependencies for the application.


## Usage

To use this application, run "npm start" command. You'll need to use Insomnia software to test POST, PUT and DELETE routes. Get routes can be tested using either browser or Insomnia.

Watch the walk through video for further reference.
